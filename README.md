# IoT Smart City Traffic Analytics Pipeline

## Overview
End-to-end Azure data engineering project using IoT, streaming, and batch analytics.

## Architecture
- Azure IoT Hub / Event Hub
- ADLS Gen2 (Raw, Processed, Curated)
- Azure Data Factory
- Azure Databricks (PySpark)
- Azure Stream Analytics
- Azure SQL / Synapse
- Power BI

## Features
- Real-time congestion alerts
- Batch + streaming ETL
- Fact & dimension modeling
- Fully version-controlled

## Tech Stack
Azure | PySpark | SQL | Power BI | GitHub
